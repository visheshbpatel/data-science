# Python Projects

This folder contains small projects built while practicing Python.

The goal of these projects is to apply Python concepts in real programs and improve problem solving through hands-on development.

## Projects

1. **Banking System (CLI)** *(In Progress)*
   A command line banking system built using Python.
   The project focuses on implementing basic banking operations and strengthening object oriented programming concepts.

2. **QuickQR** *(Completed)*
   A command line tool to generate, decode, and scan QR codes using Python and OpenCV.

   Features:

   * Generate QR codes from text or URLs
   * Decode QR codes from image files
   * Scan QR codes using a webcam

More projects will be added as I continue learning and building.

## Approach

Each project in this folder focuses on applying Python concepts to build small but practical programs.
The idea is to move from learning concepts to actually using them in real code.

## Repository Context

This folder is part of a larger repository documenting my journey while learning Python and data science.

Earlier sections focus on **Python fundamentals and advanced concepts**, and this section focuses on **building projects using those concepts**.
