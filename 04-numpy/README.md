# NumPy

This folder contains notebooks and practice code for learning NumPy, the fundamental library for numerical computing in Python.

NumPy is widely used in data science, machine learning, and scientific computing because it allows efficient operations on large datasets using arrays.

## Topics Covered

1. Introduction to NumPy
2. Creating Arrays
3. Array Attributes (shape, size, dtype)
4. Indexing and Slicing
5. Array Operations
6. Mathematical Functions
7. Broadcasting
8. Aggregation Functions (sum, mean, min, max)
9. Reshaping Arrays
10. Random Module

## Approach

The goal of this section is to understand how numerical data is handled efficiently in Python.

Instead of using traditional Python lists, NumPy arrays are used to perform operations faster and with less code.

The focus is on:
- Understanding how arrays work internally
- Practicing operations step by step
- Building intuition for vectorized computations

## Why NumPy?

NumPy forms the base for most data science libraries like Pandas, Matplotlib, and Scikit-learn.

A strong understanding of NumPy makes it easier to:
- Work with large datasets
- Perform mathematical computations efficiently
- Understand how data flows in machine learning models
