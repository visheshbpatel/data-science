# Advanced Python

This folder contains notebooks and practice code for understanding important Python concepts beyond the basics.

The focus here is on learning how Python behaves internally and how its core features are used in real programs.

## Topics Covered

1. File Handling
2. Exception Handling
3. Iterators and Generators
4. Decorators
5. Namespaces and Scopes
6. Modules and Packages
7. Working with Python Internals

## Approach

The goal of this section is to develop a deeper understanding of Python by exploring concepts that control how programs execute and how Python manages objects, functions, and scope.

These topics are important for writing cleaner code and understanding how Python libraries work internally.

## Repository Context

This folder is part of a larger repository documenting my journey while learning Python and data science.

After strengthening Python fundamentals and advanced concepts, the next step is applying them in **hands-on Python projects** and later using them with **data science libraries like NumPy and Pandas**.
