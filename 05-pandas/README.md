# Pandas

This folder contains notebooks and practice code for learning Pandas, a powerful library used for data manipulation and analysis in Python.

Pandas is built on top of NumPy and provides easy-to-use data structures like DataFrames and Series to work with structured data.

## Topics Covered

1. Introduction to Pandas
2. Series and DataFrame
3. Data Loading (CSV, Excel, etc.)
4. Data Inspection (head, tail, info, describe)
5. Indexing and Selection
6. Filtering Data
7. Handling Missing Values
8. GroupBy Operations
9. Sorting and Ranking
10. Merging and Joining DataFrames
11. Applying Functions
12. Basic Data Analysis

## Approach

The goal of this section is to learn how to work with real-world datasets.

The focus is on:
- Understanding DataFrames as tables
- Practicing data cleaning and transformation
- Performing analysis using simple operations

Instead of just learning functions, the approach is to solve small data problems step by step.

## Why Pandas?

Pandas is one of the most important libraries in data science.

It is used for:
- Cleaning messy data
- Analyzing datasets
- Preparing data for machine learning models

It provides a simple and powerful way to work with structured data efficiently.
