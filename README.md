# Data Science

A public repository documenting my journey while learning Data Science and Python.

This repository contains notes, practice notebooks, implementations, and small projects covering:
- Python Fundamentals
- Advanced Python
- Python Projects
- NumPy
- Pandas
- Data Visualization
- Data Analysis

## Learning Approach

The goal of this repository is to build strong fundamentals step by step instead of only focusing on theory.

The focus is on:
1. Understanding concepts clearly
2. Practicing through code
3. Building consistency while learning
4. Applying concepts in small projects and exercises

## Repository Structure

1. `01-python`  
   Python basics and core programming concepts

2. `02-advanced-python`  
   Advanced Python concepts and internal working

3. `03-python-projects`  
   Small projects for practical learning

4. `04-numpy`  
   Numerical computing with NumPy

5. `05-pandas`  
   Data analysis and data manipulation

6. `06-data-visualization`  
   Visualizing data using Python libraries

7. `07-data-analysis`  
   Working with datasets and extracting insights

## Purpose

This repository is mainly created for:
- Tracking my learning journey
- Practicing concepts regularly
- Building a strong base for Machine Learning and AI
- Creating a reference for future revision
