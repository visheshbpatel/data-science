# Data Visualization

This folder contains notebooks and practice code for learning data visualization in Python using Matplotlib and Seaborn.

Data visualization helps in understanding data through graphs and charts.

## Topics Covered

1. Introduction to Matplotlib  
2. Line Plots  
3. Bar Charts  
4. Scatter Plots  
5. Histograms  
6. Pie Charts  
7. Plot Customization  
8. Subplots  
9. Seaborn Basics  
10. Statistical Plots  
11. Heatmaps  
12. Pairplots  
13. Regression Plots  
14. Advanced Visualizations  

## Approach

The goal of this section is to learn how to create clear and useful visualizations from data.

The focus is on:
- Understanding different chart types
- Practicing plotting step by step
- Learning plot customization
- Visualizing real datasets

## Why Data Visualization?

Data visualization is important for:
- Understanding patterns in data
- Finding trends and insights
- Presenting information clearly
- Supporting data analysis

Matplotlib and Seaborn are widely used libraries for creating visualizations in Python.
